# BravoStorm

Website institucional desenvolvido para a empresa BravoStorm, especializada em terceirização de serviços de bombeiro civil.

O projeto foi criado como um site institucional para apresentar a empresa, seus serviços, formas de contato e informações institucionais para potenciais clientes.

## Sobre o Projeto

BravoStorm é uma empresa voltada para prestação de serviços de bombeiros civis, oferecendo profissionais treinados para atuação preventiva e emergencial em empresas, eventos e estabelecimentos que necessitam de suporte de segurança contra incêndios.

Este site foi desenvolvido com o objetivo de:

- Apresentar a empresa e sua missão
- Expor os serviços oferecidos
- Facilitar o contato com clientes e parceiros
- Criar presença digital para a marca

## Funcionalidades

- Página inicial institucional
- Apresentação da empresa
- Seção de serviços
- Informações sobre atuação de bombeiros civis
- Formas de contato
- Layout responsivo para dispositivos móveis

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias web modernas:

- HTML
- CSS
- JavaScript
- Node
- Express
- JQuery
- NodeMailer
